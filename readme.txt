# Paste Manager

## 📌 Project Overview
Paste Manager is a simple **React.js** application that allows users to create, display, and manage pastes efficiently. Users can add a title and text content, view all pastes, and delete specific ones using a search feature.

## 🚀 Features
- **Create Pastes**: Users can enter a title and text to create a new paste.
- **Display Multiple Pastes**: All created pastes are dynamically displayed on the page.
- **Delete Pastes**: Users can delete any paste individually.
- **Search & Delete by Title**: Users can search for pastes by title and delete matching ones.

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Hooks Used**: `useState`, `useEffect`, `useRef`

## 🎯 How to Run the Project
1. **Clone the repository**
   ```sh
   git clone https://github.com/yourusername/paste-manager.git
   ```
2. **Navigate to the project directory**
   ```sh
   cd paste-manager
   ```
3. **Install dependencies**
   ```sh
   npm install
   ```
4. **Start the development server**
   ```sh
   npm start
   ```
